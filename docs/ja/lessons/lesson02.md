# 第2回 シンプルな電子楽器を作ろう

デバイスアート・サウンドアートの世界を知り、micro:bit のさまざまなセンサで楽器を作ります。最後に Pure Data（Pd）に初めて触れます。

## 今回のゴール

- デバイスアート・サウンドアートの具体的な作品例を知る
- タッチセンサでドラムパッドを作る
- 電子コンパスでテルミン風楽器を作る
- Pure Data をインストールし、基本的な UI を理解する

---

## デバイスアート・サウンドアートの紹介

### デバイスアートの特徴

デバイスアートは、テクノロジーを「道具」ではなく「表現そのもの」として扱うアートです。

- **インタラクティブ性**: 鑑賞者の操作・動きに反応する
- **身体性**: 体を使った操作がアートの一部になる
- **リアルタイム性**: 入力に対してリアルタイムに音や映像が変化する

### 有名な作品例

- **テルミン**（1920年）: 手を近づけるだけで演奏できる世界初の電子楽器
- **TENORI-ON**（岩井俊雄、2007年）: LED のグリッドに触れて音楽を作るデバイス
- **Reactable**（2003年〜）: テーブル上にブロックを置いて音を作るインターフェース
- **Sound of Honda / Ayrton Senna 1989**: F1 の走行データを音に変換したサウンドインスタレーション

### サウンドアートのアプローチ

- **環境音の活用**: 日常の音をアートに昇華する
- **身体の拡張**: デバイスで体の動きを音に変換する
- **データのソニフィケーション**: データを音で表現する

::: tip 考えてみよう
「身近にある動き」を音に変換するとしたら、どんな動きを選びますか？ どんな音にマッピングしますか？
:::

## Step 1: タッチセンサでドラムを叩こう

micro:bit V2 のタッチセンサ（ロゴマーク）や端子（P0, P1, P2）を使って、ドラムパッドを作ります。

### タッチピンの使い方

micro:bit の下部にある金色の端子（ピン）は、タッチセンサとして使えます。

- **P0**: 大きな端子（左から1番目）
- **P1**: 大きな端子（左から2番目）
- **P2**: 大きな端子（左から3番目）
- **GND**: アース端子（右端）

::: warning 注意
タッチセンサを使うには、片手で **GND** 端子に触れながら、もう片方の手（または指）で P0〜P2 に触れる必要があります。ワニ口クリップを GND に接続してアルミホイルに繋げると便利です。
:::

### 作り方

1. MakeCode で新しいプロジェクトを作成する
2. 以下のようにプログラムを組む:

**P0 に触れたとき → バスドラムの音**

```
端子 P0 がタッチされたとき:
  音楽「バスドラム」を鳴らす
  LEDに "X" を表示
```

**P1 に触れたとき → スネアの音**

```
端子 P1 がタッチされたとき:
  音楽「スネア」を鳴らす
  LEDに "+" を表示
```

**P2 に触れたとき → ハイハットの音**

```
端子 P2 がタッチされたとき:
  音楽「ハイハット」を鳴らす
  LEDに "." を表示
```

3. 各端子に異なるドラム音を割り当てる
4. 「音楽」→「サウンドを鳴らす」でプリセットのドラム音を選択する

### 音の割り当てを工夫する

- V2 の **ロゴタッチ** にもシンバルの音を割り当てる
- ボタン A、ボタン B にも別の音を割り当てると、最大6種類の音が出せる

## Step 2: 電子コンパスでテルミンを作ろう

micro:bit の電子コンパス（地磁気センサ）を使って、向いている方角で音の高さが変わる楽器を作ります。

### コンパスセンサの説明

- micro:bit を水平に持ち、回転させると **方位角** が取得できる
- 値の範囲: **0〜359**（度）
- 北 = 0°、東 = 90°、南 = 180°、西 = 270°

::: warning キャリブレーション
コンパスを初めて使うとき、micro:bit の画面に「TILT TO FILL SCREEN」と表示されます。micro:bit を傾けて全ての LED を点灯させると、キャリブレーション（校正）が完了します。
:::

### 方角から周波数へのマッピング

方位角（0〜359）を音の周波数に変換します。

```
ずっと:
  方位 = コンパスの方角
  周波数 = 数値をマップする(方位, 0, 359, 200, 2000)
  その周波数で音を鳴らす
```

1. 「入力」→「その他」→ **「コンパスの方角」** ブロックを使う
2. 「計算」→ **「数値をマップする」** で、0〜359 を 200〜2000（Hz）に変換する
3. 「音楽」→ **「音を鳴らす」** に変換した値を渡す

### 応用アイデア

- 方角によって異なる音階（ドレミファソラシド）を鳴らす
- 特定の方角に向いたときだけ音が鳴るようにする
- ボタンでオクターブを切り替える

## Step 3: Pure Data に触れてみよう

### Pure Data（Pd）とは

**Pure Data（Pd）** は、ビジュアルプログラミングで音響合成やマルチメディア処理を行うためのオープンソースソフトウェアです。

- **開発者**: Miller Puckette（1990年代〜）
- **特徴**: オブジェクト（箱）をパッチコード（線）で接続してプログラムを作る
- **用途**: 電子音楽、サウンドインスタレーション、インタラクティブアート
- **無料**: オープンソースで誰でも使える

### Pd の種類

| 名称 | 説明 |
|------|------|
| **Pd vanilla** | Miller Puckette が開発する公式版（最小限の機能） |
| **Pd-extended** | 多くのライブラリを同梱した拡張版（開発終了） |
| **Plugdata** | モダンな UI の Pd 互換環境（VST プラグインとしても動作） |

この授業では **plugdata** を使用します。

### plugdata について

**[plugdata](https://plugdata.org/)** は、Pure Data をベースにしたモダンなビジュアルプログラミング環境です。Pd vanilla と互換性がありながら、より使いやすいインターフェースを備えています。

| 特徴 | 説明 |
|------|------|
| **モダンな UI** | ダークモード対応の見やすいインターフェース。オブジェクトのオートコンプリートやドラッグ＆ドロップに対応 |
| **Pd 互換** | Pd vanilla のパッチをそのまま開ける。このサイトで紹介する Pd の操作・オブジェクトはすべて plugdata でも使える |
| **ライブラリ内蔵** | ELSE や cyclone など、授業で使うライブラリがあらかじめ内蔵されており、追加インストールが不要 |
| **VST/AU プラグイン** | DAW（GarageBand、Ableton Live 等）のプラグインとしても動作する |
| **クロスプラットフォーム** | Mac / Windows / Linux 対応 |

::: tip 大学 PC にはインストール済みです
plugdata は大学の PC にすでにインストールされています。自分の PC にもインストールしたい場合は、[plugdata 公式サイト](https://plugdata.org/) からダウンロードしてください。
:::

::: warning Pd vanilla との違い
操作方法やオブジェクトの名前は Pd vanilla と同じですが、ショートカットキーや見た目が若干異なります。このサイトのスクリーンショットは Pd vanilla のものですが、plugdata でも同じ操作で同じ結果が得られます。
:::

## Pure Data の基本 UI

plugdata（または Pd vanilla）を起動して、基本的な画面構成を確認しましょう。

- **パッチウィンドウ**: プログラム（パッチ）を作成する作業スペース
- **オブジェクト**: `Ctrl+1`（Mac: `Cmd+1`）で新しいオブジェクトを配置できる
- **メッセージ**: `Ctrl+2`（Mac: `Cmd+2`）でメッセージボックスを配置できる
- **編集/実行モードの切替**: `Ctrl+E`（Mac: `Cmd+E`）

新しいパッチを作成し、オブジェクトやメッセージを配置してみましょう。次回の授業で本格的に使い始めます。
